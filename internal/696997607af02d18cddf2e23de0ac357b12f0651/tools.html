Tips

  • Git: undo last commit
    • git reset --hard HEAD~1 (completely remove all changes in the last commit)
    • git reset HEAD~1 (set HEADER 1 step back and keep changes)
    • git reset --soft HEAD~1 (set HEADER 1 step back and keep changes and indexings)
  • Set heap size up to 5120 in Crashplan
    • double click home icon
    • java mx 5120 restart
  • To show installed files for a package
    • rpm -ql [package]
  • To mount usb HD with exfat format
    • install exfat-utils and fuse-exfat
    • check out link
    • mount.exfat ...
  • Access $HOME placed on a server via NFS
    • setsebool -P use_nfs_home_dirs 1
  • Setup autostart
    • sudo systemctl enable XXXd
  • Use 'eog' to open images
  • To show path of a running job
    • pwdx [PID]
  • Amber MD with distance restraints
    • Distance restraint is NOT re-imaged!
    • So, if restart file has separate two chains where the restraint is applied, do the re-image manually before resuming the simulation
    • Alternatively, we should TURN OFF wrapping (iwrap=0) and write both trajectory and restart in binary (ioutfm=1, ntxo=2)
  • Use photoshop to make a movie
    • File > Scripts > Load Files into Stacks... (This option will add all your image in as one file with separate layers) and add all the images you want in the animation.
    • Open the Animation Palette under: Window > Timeline.
    • Create Frame Animation.
    • click on the Flyout Menu on top right corner of the animation (or layer) window and select (Make) Frames from Layers. NOTE: If your frames are backwards, click the bottom left icon to of the animation window get into "frame animation mode", select all the square frames then click the flyout again and reverse frames.
    • Adjust the timing of each frame by selecting the 0.1 sec. on the frame.
    • File > Export > Render Video. Among the many options you can save as a movie for Flv, IPhone, IPod, Quick Time, MPEG4 and so on. I recommend using compression type "H.264".
  • In Bash (e.g., CentOS 7), sometimes `ls XXX.[a-c].YYY` does not produce what you expect. Have to spell it out like `ls XXX.[abc].YYY`.
